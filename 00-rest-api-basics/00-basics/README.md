# REST API Basics (Python)

Dieses Projekt demonstriert die grundlegende Integration von REST APIs mit Python. 
Es deckt die wichtigsten CRUD-Operationen (Create, Read, Update, Delete) ab.

## Features
- **GET**: Daten von einer API abrufen.
- **POST**: Neue Ressourcen erstellen.
- **PUT**: Bestehende Daten aktualisieren.
- **DELETE**: Daten entfernen.
- **Logging**: Alle Aktionen werden mit Zeitstempel in eine Datei geschrieben.
